# 사망

상태: 시작 전
작성일시: 2022년 1월 16일 오후 9:58

### UDP

```jsx
프로토콜: PD

message: 닉네임

처리내용:
1. 몬스터가 공격시 플레이어가 사망한다면 클라이언트로 부터 PD 프로토콜 요청을 받는다.
2. 유저 DB에서 동일한 닉네임을 조회한다.
3. 유저 정보를 삭제한다.
4. 참여자 배열에서 유저 객체를 제거한다.
5. 클라이언트로 true값을 넘겨준다.
```

## 검증

input:

```jsx
path: 127.0.0.1:9000
message:{cmd:PD,nick:im warrior}
```

output: 

```jsx
{cmd:PD,ture}
```
